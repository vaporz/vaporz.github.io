Kift APIs

Kift provides two sets of APIs: "InfoSync APIs" and "Command APIs".

"InfoSync APIs" pull infos from the game. "Command APIs" send commands to control vessels/parts/game.

Command APIs are devided into three categories according to the target.

  • InfoSync APIs
    • Real time snapshot
    • GetInfo APIs
  • Command APIs
    • Vessel Control APIs
    • Part Control APIs
    • Game Control APIs

InfoSync APIs

Snapshot

Read data at real time(approximately), useful when you want to monitor any data, such as height, speed, etc.

SnapshotResponse snapshot(1:SnapshotRequest req)

GetInfo APIs

Get data from current game, such as current vessel, the part list of a vessel, etc.

list listVessel()

KVessel currentVessel()

KVessel getVessel(1:string vesselId)

list getVesselParts(1:string vesselId)

list getVesselPartsById(1:string vesselId, 2:list partIds)

list getVesselPartsByType(1:string vesselId, 2:PartType type)

list getPartsById(1:list partIds)

list listCelestialBody()

KCelestialBody getCelestialBody(1:string name)

GetTargetVesselOrbitInfoResponse getTargetVesselOrbitInfo(1:string vesselId, 2:string targetVesselId)

GetTargetBodyOrbitInfoResponse getTargetBodyOrbitInfo(1:string vesselId, 2:string targetBodyName)

GetTargetVesselOrbitInfoFromManeuverNodeResponse getTargetVesselOrbitInfoFromManeuverNode(1:string vesselId, 2:double ut, 3:string targetVesselId)

GetTargetBodyOrbitInfoFromManeuverNodeResponse getTargetBodyOrbitInfoFromManeuverNode(1:string vesselId, 2:double ut, 3:string targetBodyName)

AttitudeInfo getVesselAttitudeInfo(1:string vesselId)

AttitudeInfo getPartAttitudeInfo(1:i64 partId)
